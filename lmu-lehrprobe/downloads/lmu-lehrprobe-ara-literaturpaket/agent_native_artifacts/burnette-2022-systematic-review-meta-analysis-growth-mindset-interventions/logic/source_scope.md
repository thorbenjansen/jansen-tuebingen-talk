# Source Scope

This file states what the source can safely support in later wiki queries and where a raw-source check is still required.

## Strong Uses

- Use as a review or meta-analysis connected to Educational Annotation, Formative Feedback, Interventions, Meta-Analysis, Motivation, Self-Beliefs.
- Use for concept-level retrieval and synthesis through `logic/concepts.md` and `logic/theories.md`.
- Use for broad claims that match `logic/claims.md`, `logic/method_and_design.md`, and `evidence/README.md`.

## Weak Uses

- Do not cite exact numeric effects, page-specific claims, model coefficients, or sample details without checking the raw PDF or high-quality Markdown sidecar.
- Do not treat inferred theory links as explicit citations until the citation context has been checked.
- Do not use this source outside its documented population, design, source material, and outcome boundaries.

## Transfer Boundaries

- For writing, feedback, assessment, AI, motivation, and learning-process questions, use this source only where the concept links above match the claim being made.
- Boundary or adjacent-domain sources should support analogy, theory, or method claims, not direct empirical claims about student AI writing unless the source itself studies that setting.

## Follow-Up For Citation-Grade Use

- Verify the exact theory citations and key empirical numbers against the PDF/Markdown before using this source in a manuscript.
