# Method And Design

## Design Type

Meta-analysis of learning/study-skills intervention studies.

## Source Material

51 studies of interventions designed to improve student learning through study, learning, self-management, or affective skills.

## Method Workflow

- Classifies interventions by structural complexity using the SOLO model.
- Distinguishes near-transfer and far-transfer outcomes.
- Synthesizes intervention features associated with stronger learning effects.

## Population Or Study Context

The population and context are inherited from the included studies. For synthesis, use this ARA as review/meta-analytic evidence rather than as a single primary classroom sample.

## Citation-Grade Verification Needed

- Verify exact inclusion counts, effect sizes, and moderator coefficients in the PDF before quoting them.
- Use the source PDF and raw Markdown sidecar for exact statistical reporting.
