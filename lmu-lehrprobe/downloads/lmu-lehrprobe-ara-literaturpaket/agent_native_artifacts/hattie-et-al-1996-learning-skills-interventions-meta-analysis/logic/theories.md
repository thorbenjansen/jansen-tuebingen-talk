# Theories

## Theory Inventory

### SOLO Taxonomy

- Theory concept: [[SOLO Taxonomy]]
- Role in this source: Classifies structural complexity of learning outcomes and intervention demands.
- How the source uses it: Used as an analytic coding framework.
- APA anchor: Biggs & Collis (1982), cited by Hattie et al. (1996).

### Situated Cognition

- Theory concept: [[Situated Cognition]]
- Role in this source: Learning strategies transfer better when practiced in authentic or domain-relevant task contexts.
- How the source uses it: Used as a core interpretation of the meta-analytic findings.
- APA anchor: Situated-cognition tradition, cited through Hattie et al. (1996).

### Self-Regulated Learning

- Theory concept: [[Self-Regulated Learning]]
- Role in this source: Connects self-management, metacognition, and learner activity to durable learning.
- How the source uses it: Used as a conceptual umbrella for many intervention components.
- APA anchor: Hattie et al. (1996).

## APA Reference For This Source
