# Source Scope

## Included

- Corrected source identity, author/year/title, and source role.
- Source-specific claims, method/design profile, concepts, theories, and limits.
- A source note connected to concept pages for Obsidian graph use.

## Excluded Or Deferred

- Exhaustive page-by-page claim mapping.
- Full reference-list audit.
- Long verbatim excerpts.

## Boundary

Use for construct clarification; exact moderator/results details should be checked for manuscript claims.
