# Methods Passage Summary

The review used a multimodal search strategy and included first-order meta-analyses that reported at least one quantitative effect size between a variable and a self-belief construct for children/adolescents in K-12 education.

The authors double-coded meta-analyses using a coding scheme covering effect sizes, variable categories, moderators, and transparency/reproducibility criteria. First-order effect sizes were converted to standardized mean differences. Second-order random-effects meta-analyses used robust variance estimation to handle dependent effect sizes.

Source location: pp. 1291-1298.
