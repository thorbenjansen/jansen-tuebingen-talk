# Level 3 Study Profile

| Field | Value |
|---|---|
| Source type | Second-order meta-analytic review |
| Domain | Educational psychology; self-beliefs; K-12 education |
| Population | Children and adolescents in K-12 education |
| Evidence base | 105 first-order meta-analyses; 493 first-order effect sizes |
| Scale | More than 8,500 primary studies and more than 16 million children/adolescents |
| Main method | Random-effects second-order meta-analysis with robust variance estimation |
| Main outcome | Self-beliefs, broadly construed |
| Main comparison | Personal characteristics versus interventions |
| Methodological review | Transparency/reproducibility criteria for the included meta-analyses |
| Main caution | Primarily descriptive/correlational, not causal |
