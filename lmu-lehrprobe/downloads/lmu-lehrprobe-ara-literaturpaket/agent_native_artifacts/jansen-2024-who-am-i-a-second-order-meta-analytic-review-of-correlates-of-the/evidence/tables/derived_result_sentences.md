# Results Passage Summary

The paper included 105 meta-analyses and 493 first-order effect sizes. The overall second-order estimate was SMD = 0.40 [0.34, 0.46].

The main results were:

- Personal characteristics: SMD = 0.50 [0.42, 0.59].
- Interventions: SMD = 0.27 [0.19, 0.34].
- Achievement: SMD = 0.66 [0.57, 0.75].
- Noncognitive variables: SMD = 0.67 [0.54, 0.80].
- Cognitive ability: SMD = 0.30 [0.15, 0.46].
- Background variables: SMD = 0.21 [0.17, 0.25].
- Direct self-belief interventions: SMD = 0.52 [0.48, 0.55].

Situational characteristics were reviewed narratively because the evidence was too sparse for meaningful integrated effect-size comparison.

Source location: pp. 1298-1302.
