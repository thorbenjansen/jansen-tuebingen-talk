# Retrieval Heuristics

- For headline numbers, use `evidence/tables/level3_numeric_results.md`.
- For conceptual meaning, use `logic/concepts.md`.
- For causal boundaries, use `logic/source_scope.md`.
- For intervention claims, check whether the claim refers to all interventions, personal-outcome interventions, or direct self-belief interventions.
- For motivation-related questions, connect noncognitive variables and the Jansen et al. (2022) motivation review.
- Avoid using this source as direct evidence about writing unless the link is through students' motivation, achievement, self-beliefs, or intervention design.
